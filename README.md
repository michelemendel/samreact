# ucp-light

### Install

All script are in package.json.

* clone
* npm install
* npm start


### Architecture

* [React](https://facebook.github.io/react/)
* [Sam-pattern](http://sam.js.org/).

`V = S( vm(M.present(A(data))), nap(Model) )`
